# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-19

### Added
- Rocket Management API endpoints for create, list, get by id, update, and delete operations.
- Validation for rocket `name`, `range` (`suborbital`, `orbital`, `moon`, `mars`) and `capacity` (1 to 10).
- End-to-end tests covering all rocket acceptance criteria from `specs/rockets.spec.md`.

### Changed
- Set project version metadata to `0.1.0` for release preparation.
